## 게임 설명
### 조작법:
- 이동 : WASD
- 달리기 : Lshift
- 점프 : Space
- 인벤토리 : Tab
- 아이템 사용 : 마우스 우클릭
- 아이템 정보 : 마우스 가운데

### 구현 내용
#### 플레이어
- **WASD를 이용하여 (앞,뒤,좌,우)이동**
- **Space를 이용하여 점프**
- **LShift를 사용하여 이동속도 증가**
